## 💩 프로젝트 명
PoopTube

## 📕 프로젝트 설명
저희조의 시그니쳐인 ‘응가(Poop)’와 미디어를 보며 재밌어서 ‘풉풉’ 하며 웃는 모습의 이중적인 의미를 담은 프로젝트 입니다.
유튜브 API를 이용하여 가장 인기있는 동영상/ 카테고리별 영상 목록을 보여주고, 검색까지 할 수 있는 어플입니다.
